# PhotoStudio CRM Software 

A modern Customer Relationship Management (CRM) system designed for photo studios to manage client interactions, bookings, project updates, and photo delivery — all in one place with a clean glassmorphism UI.

🚀 Features

👨‍💼 Admin Dashboard

📊 Analytics (Revenue, Bookings, Clients, Deliveries)

📅 Booking & Calendar Management

👥 Client Management

📸 Project & Photo Management

🔔 Notifications & Alerts

💰 Payment Tracking

👤 Client Panel

📌 Book photo sessions

📊 Track project status (Shoot → Editing → Delivery)

🖼️ View & download photos

🔔 Receive real-time updates

💳 View payment details

🎨 UI/UX Design

✨ Glassmorphism (frosted glass effect)

🌈 Soft pastel color palette (lavender, blue, mint, peach)

🧊 Blur backgrounds & transparent cards

🔲 Rounded corners & soft shadows

📱 Fully responsive design

🧭 Clean and minimal SaaS layout

🧩 Modules

Authentication (Admin & Client)

Booking Management

Client Management

Project Tracking

Notifications System

Photo Gallery Delivery

Payment & Invoice Management

Analytics Dashboard
